# Repair Relay

Repair Relay is a web-based workflow management system for collision repair shops that improves communication, repair visibility, parts tracking, and customer updates from check-in through closeout.

## Problem Statement

Collision repair shops often rely on spreadsheets, phone calls, text messages, and handwritten notes to track repairs, parts, approvals, and customer communication. These disconnected methods create miscommunication between office staff, estimators, technicians, and customers, which leads to delays, inconsistent updates, and poor workflow visibility.

## Solution

Repair Relay is a web-based platform that centralizes the repair process into one visual workflow system. It provides department-specific boards for technicians, estimators, paint, detail, and parts while also improving customer communication through clearer repair updates, notifications, and status tracking.

## Key Features

- Department workflow boards for technicians, paint, detail, and estimators
- Repair status tracking with priority, parts, and blocker visibility
- Parts arrival alerts and workflow notifications
- Customer communication updates through text-friendly status messaging
- Post-completion file review and closeout tracking
